Strategy Pillars

  • Governed innovation with explicit exceptions and controls
  • Measurability-first: KPIs and telemetry for each pilot
  • Defense-in-depth: private endpoints, policy gateway, logging, redaction, RBAC
  • Interoperability: MCP as the integration layer; policy mediation otherwise
  • Vendor resilience: BYOK/BYOM behind policy abstractions; avoid lock-in

90-Day Plan

  1. Phase 1 (Weeks 0–2) — Optimize GitHub Copilot/Agent Mode; baseline KPIs
  2. Phase 2 (Weeks 2–6) — Enablements beyond MCP: policy gateway, telemetry, advanced Copilot workflows, code search/RAG, review/test, knowledge integrations
  3. Phase 3 (Weeks 4–10) — Differentiator pilots: CLI agents, BYOK personal agents, spec-first, migration/refactor, review/test
  4. Weeks 8–12 — Decision/readout: normalized metrics, TCO, security assessment, staged rollout
